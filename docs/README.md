# CivicPulse NYC

**Prototype** — AI-powered civic news classifier and digest generator. Probably one user... me!

[View live →](https://stephendongg.github.io/local-news-organizer/nyc/)

## How it works

Fetches NYC news → Classifies with gpt-5.4 → Filters by importance → Generates digest

Uses a custom taxonomy adapted from the FCC's Critical Information Needs framework for civic categorization.

Automated via GitHub Actions on pushes to `main`, with manual runs available from the Actions tab.
